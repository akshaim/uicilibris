The constructor.

Parameters:
argsa list of filenames, or a single URL. When a single URL, it is meant to be the address of a wikimedia page which contains a series of other addresses of the same mediawiki in mediawiki syntax.
isattyis True when data do not come from the standard input
isUrlis True to force the initialization, by considering arg as a single url which is supposed to be a normal wiki page
reportif True, messages are emitted to sys.stderr; if it is callable, it is invoked with the same messages
parentpointer to the main window
latexReadyParamthe parameter for a signal to be emitted upon completion of the process
oldWiki2beamerwhen True, uses the modes of wiki2beamer
